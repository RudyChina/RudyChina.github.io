# WebSocket实时推送功能

业务场景:

行情数据实时push到Redids中的list队列中，客户端需要实时刷新行情数据。客户端发送过滤条件到服务端，服务端将pop出来的数据进行过滤，实时的向客户端推送新的行情数据。类似上图

# WebSocket

  • Http协议存在一个弊端：通信只能由客户端发起，无法做到服务端主动向客户端推送数据。

  • 所以在早期就有了类似Ajax轮询的做法

    • 每隔一段时间向服务端发起请求，拉取新的数据，但是这种做法效率比较低。需要不停的建立Http连接，浪费网络资源，或者保持Http连接始终打开。

  • 而WebSocket的出现解决了这个问题，他是基于TCP上的一种全双工通信，是一种持久化的协议。服务端和客户端可以随时发起通信。

# WebSocket拦截器

  • 一般在拦截器里做握手前后的逻辑处理。只需要继承HttpSessionHandshakeInterceptor类，实现对应方法。
  • 比较重要的一点是：在WebSocket连接里是不好拿到HttpSession，因此可以在拦截器中的ServerHttpRequest去拿到服务端的http请求，从而获取当前http会话。拿到会话后缓存起来，可以缓存成session注册表，也可以用本地线程ThreadLocal绑定。
  • 拦截器代码如下：在拦截器里我主要是获取到当前用户的信息（身份信息，会话信息）。
/**
 * WebSocket拦截器
 * @author 周鸿宇
 */
public class WebSocketInteceptor extends HttpSessionHandshakeInterceptor {
    private static Logger log = Logger.getLogger(WebSocketInteceptor.class);

    /**
     * WebSocket握手前处理
     * @throws Exception
     */
    @Override
    public boolean beforeHandshake(ServerHttpRequest request, ServerHttpResponse response, WebSocketHandler wsHandler, Map<String, Object> attributes) throws Exception {
        //获取系统用户ID，处理器层用作会话令牌
        ServletServerHttpRequest req = (ServletServerHttpRequest) request;
        UserSession userSession = (UserSession) 					  req.getServletRequest().getSession().getAttribute(SysConfigUtil.getUserSessionName());
        if (userSession == null) {
            //httpSession为空或过期断开Socket连接
            return false;
        }
        String uid = userSession.getId();
        if (StringUtils.isNotEmpty(uid)) {
            attributes.put("USER_SESSION",uid);
        }
        return super.beforeHandshake(request, response, wsHandler, attributes);
    }

    /**
     * WebSocket握手后处理
     * @param ex
     */
    @Override
    public void afterHandshake(ServerHttpRequest request, ServerHttpResponse response, WebSocketHandler wsHandler, Exception ex) {
        super.afterHandshake(request, response, wsHandler, ex);
    }
}

# WebSocket消息处理器

  • WebSocket发送数据的格式有两种：文本帧数据，二进制帧数据。我用的是文本帧。

  • 还有一点需要注意的是在多线程并发情况下，会存在分包、粘包的现象。也就是数据被截断，一条数据可能并发下成了数组。解决方案是指定长度和数据末尾加分隔符

  • 在处理器中主要有三个事件：afterConnectionEstablished连接建立事件，handleMessage消息处理事件，afterConnectionClosed连接断开事件。

    • 在连接建事件将userId和WebSocketSession通道绑定。
    • 在消息处理事件将WebSocket管道和过滤条件绑定。（业务场景是一个用户可以开启多个通道，查看多组数据），在第一个通道建立的时候就开启新的线程开始从Redsi队列Pop数据。
    • 连接断开事件主要用来刷新查询条件缓存和WebSocketSession注册表缓存

  • 处理器代码如下：

/**
 * WebSocket文本帧数据处理器
 * @author zlf
 * @date 2020-07-11 16:40
 */
@Component
public class TrendSocketHandler extends TextWebSocketHandler implements WebSocketHandler {
    private static final Logger log = Logger.getLogger(TrendSocketHandler.class);
    /** 首页走势图pop队列名 **/
    private static final String INDEX_QUEUE = DBConfigUtil.get("index.queue");

    @Resource(name = "defaultRedisTemplate")
    private StringRedisTemplate redisTemplate;


    /** Socket Session缓存 user_Id和SocketSession绑定 */
    public static  Map<String, CopyOnWriteArraySet<WebSocketSession>> SOCKET_SESSION_REGISTRY_MAP;
    
  	/** 查询条件缓存 SocketSessionID 和查询条件绑定*/
    public static  Map<String,String> TREND_BIND_PIPE_MAP;
    public static  Map<String,List<String>> DASH_BIND_PIPE_MAP;
    
  	/** 上一条走势图数据和表盘数据绑定 socketSession,做行情去重处理 **/
    private static Map<String,String> LAST_TREND_MAP;
    private static Map<String,String> LAST_DASH_MAP;
		/** 是否新起一个线程 **/
		private static volatile Boolean ThreadOpenFlag = false;

    //初始化静态变量
    static{
        SOCKET_SESSION_REGISTRY_MAP = new ConcurrentHashMap<>();
        TREND_BIND_PIPE_MAP = new ConcurrentHashMap<>();
        DASH_BIND_PIPE_MAP = new ConcurrentHashMap<>();
        LAST_TREND_MAP = new ConcurrentHashMap<>();
        LAST_DASH_MAP = new ConcurrentHashMap<>();
    }


    /** 建立连接事件 */
    @Override
    public void afterConnectionEstablished(WebSocketSession session) throws Exception {
        Map<String, Object> map = session.getAttributes();
        String userId = (String) map.get("USER_SESSION");
        //TODO WebSocketSession绑定user_id
        if (SOCKET_SESSION_REGISTRY_MAP.containsKey(userId)) {
            Set<WebSocketSession> SocketSessionSet = SOCKET_SESSION_REGISTRY_MAP.get(userId);
            SocketSessionSet.add(session);
        }else {
            CopyOnWriteArraySet<WebSocketSession> set = new CopyOnWriteArraySet<>();
            set.add(session);
            SOCKET_SESSION_REGISTRY_MAP.put(userId,set);
        }
        log.info("==================WebSocket会话建立,{PIPE_ID:"+session.getId()+"}==============================");
    }

    /**
     * 接收消息
     * @param webSocketSession
     * @param webSocketMessage
     * @throws Exception
     */
    @Override
    public void handleMessage(WebSocketSession webSocketSession, WebSocketMessage<?> webSocketMessage) throws Exception {
        //获取当前用户ID
        Map<String, Object> map = webSocketSession.getAttributes();
        String jsonData = webSocketMessage.getPayload().toString();
        System.out.println("请求数据:"+jsonData);
        //客户端请求数据
        ReqWebSocket reqModel = JsonUtils.jsonStrToObj(jsonData, ReqWebSocket.class);
        //处理首页走势图的实时数据刷新
        List<String> dashIdList = reqModel.getGroup_ids();
        String trendId = reqModel.getGroup_id();
        //刷新查询条件。
        TREND_BIND_PIPE_MAP.put(webSocketSession.getId(),trendId);
        DASH_BIND_PIPE_MAP.put(webSocketSession.getId(),dashIdList);
        if (!ThreadOpenFlag) {
            IndexThreadPool.sendTask(()->{
                ThreadOpenFlag = true;
                pushMessageOnTime(SOCKET_SESSION_REGISTRY_MAP);
            });
        }
    }

    /**
    * @Description: 处理首页走势图实时刷新的数据（包括仪表盘，走势图）
    * @Param: dashIdList:仪表盘需要展示的对应合约组的数据
    *         trendId:走势图对应需要展示的数据
    *         ts:走势图分时条件
     */
    private void pushMessageOnTime(Map<String, CopyOnWriteArraySet<WebSocketSession>> pipeMap) {
        try {
            //从Redsi中Pop数据。
            while (true) {
                //新的会话建立，直接结束线程，归还线程池;
                String queData = redisTemplate.opsForList().leftPop(INDEX_QUEUE);
                if (queData == null||SOCKET_SESSION_REGISTRY_MAP.isEmpty()) {
                    continue;
                }
                //向所有管道推送数据
                for (Map.Entry<String,CopyOnWriteArraySet<WebSocketSession>> sessionSet :pipeMap.entrySet()){
                    for (WebSocketSession webSocketSession : sessionSet.getValue()) {
                        //获取管道的查询条件
                        String trendId = TREND_BIND_PIPE_MAP.get(webSocketSession.getId());
                        List<String> dashIdList = DASH_BIND_PIPE_MAP.get(webSocketSession.getId());
                        if (trendId != null && dashIdList != null) {
                            pushToAllSocketClient(trendId, dashIdList, queData, webSocketSession);
                        }
                    }
                }
                queData = null;
            }
        } catch (Exception ex) {
            ThreadOpenFlag=false;
            log.error("websocket推送异常",ex);
        }
    }

    /**
    * 向所有管道推送对应的数据
    */
    private void pushToAllSocketClient(String trendId,List<String> dashIdList,String queData, WebSocketSession webSocketSession) throws Exception {
            if (queData.contains("\"tp\": \"i\"")) {
                //走势图数据4
                TrendWS trend = JsonUtils.jsonStrToObj(queData, TrendWS.class);
                if (Objects.equals(trendId, trend.getId())) {
                    if (Objects.equals(trend.toString(), LAST_TREND_MAP.get(webSocketSession.getId()))) {
                        return;
                    }
                    LAST_TREND_MAP.put(webSocketSession.getId(),trend.toString());
                    byte[] bytes = trend.toString().getBytes();
                    TextMessage message = new TextMessage(bytes);
                    sendMessageToClient(message, webSocketSession);
                }
            } else {
                //仪表盘数据
                DashBoardWS dashBoard = JsonUtils.jsonStrToObj(queData, DashBoardWS.class);
                if (dashIdList.contains(dashBoard.getId())) {
                    if (Objects.equals(dashBoard.toString(), LAST_DASH_MAP.get(webSocketSession.getId()))) {
                        return;
                    }
                    LAST_DASH_MAP.put(webSocketSession.getId(),dashBoard.toString());
                    byte[] bytes = dashBoard.toString().getBytes();
                    TextMessage message = new TextMessage(bytes);
                    sendMessageToClient(message, webSocketSession);
                }
            }
    }

    /**
     * 消息处理器
     * @param message 文本帧数据
     */
    public void sendMessageToClient(TextMessage message,WebSocketSession session){
        if (session != null && session.isOpen()) {

                IndexThreadPool.sendTask(()->{
                    try {
                        session.sendMessage(message);
                    } catch (IOException e) {
                        log.error("推送异常", e);
                        e.printStackTrace();
                    }
                });
        }
    }

        /**
         * @Description: 连接断开事件
         */
        @Override
        public void afterConnectionClosed (WebSocketSession webSocketSession, CloseStatus status) throws Exception {
            Map<String, Object> map = webSocketSession.getAttributes();
            String userId = (String) map.get("USER_SESSION");
            //刷新缓存
            CopyOnWriteArraySet<WebSocketSession> socketSessionSet = SOCKET_SESSION_REGISTRY_MAP.get(userId);
            socketSessionSet.stream().forEach(session->{
                if (webSocketSession == session) {
                    //刷新管道的查询缓存
                    TREND_BIND_PIPE_MAP.remove(webSocketSession.getId());
                    DASH_BIND_PIPE_MAP.remove(webSocketSession.getId());
                    //移除去重条件
                    LAST_TREND_MAP.remove(webSocketSession.getId());
                    LAST_DASH_MAP.remove(webSocketSession.getId());
                    //移除管道
                    socketSessionSet.remove(session);
                }
            });
            log.info("=================webSocket会话断开，踢出{PIPE_ID:"+webSocketSession.getId()+"}===================");
            super.afterConnectionClosed(webSocketSession, status);
        }
}

# WebSocket配置类

  • 这个就是一个WebSocket配置类，主要用来配置上面的拦截器，消息处理器
/**
 * WebSocket配置类
 * @author 周鸿宇
 */
@Configuration
@EnableWebMvc
@EnableWebSocket
public class WebSocketConfiguration implements WebSocketConfigurer {

    /**
     * 配置WebSocket处理器
     * 官方文档：https://docs.spring.io/spring/docs/5.0.0.BUILD-SNAPSHOT/spring-framework-reference/html/websocket.html
     * @param webSocketHandlerRegistry WebSocket注册器
     */
    @Override
    public void registerWebSocketHandlers(WebSocketHandlerRegistry webSocketHandlerRegistry) {
        webSocketHandlerRegistry
                /** 在指定的URL路径上配置WebSocketHandler。*/
                .addHandler(trendSocketHandler(), "/ws/socketServer")
                .addInterceptors(webSocketInteceptor())
                /** 允许跨域请求() */
                .setAllowedOrigins("*");
    }

    /**
     * 走势图webSocket处理器注册
     * @return 处理器实例
     */
    @Bean
    public TrendSocketHandler trendSocketHandler(){
        return new TrendSocketHandler();
    }
  
  /**
     * 走势图webSocket拦截器注册
     * @return 处理器实例
     */
    @Bean
    public WebSocketInteceptor webSocketInteceptor(){
        return new WebSocketInteceptor();
    }
}

# 线程池

  • 根据阿里巴巴开发手册规范，自定义线程池异步处理消息。
  • 这里解释下线程池中的参数：
    • corePoolSize：线程池默认大小；
    • maximumPoolSize：当任务比较多的时候，线程数已达默认设置阈值，会新建线程，最多只能创建maximumPoolSize个线程。
    • keepAliveTime：当任务比较少的时候，超过默认大小的空闲线程会在设置的这个时间后被回收。
    • TimeUnit.MILLISECONDS：时间单位
    • Queue：队列（这里我使用的是阻塞队列）
    • 拒绝策略
/**
 * @Description: 首页多线程推送WebSocket线程池
 * @Author: 周鸿宇
 */
@Component
public class IndexThreadPool {

    private static Integer corePoolSize;
    private static Integer maximumPoolSize;
    private static Long keepAliveTime;

    static{
        corePoolSize = Integer.valueOf(DBConfigUtil.get("index.thread.corePoolSize"));
        maximumPoolSize = Integer.valueOf(DBConfigUtil.get("index.thread.maximumPoolSize"));
        keepAliveTime = Long.valueOf(DBConfigUtil.get("index.thread.keepAliveTime"));
    }

    /**
    * Jps,JStack可查看对应线程状态
    * @Author: 周鸿宇
    */
    private static ThreadFactory factory = new ThreadFactoryBuilder().setNameFormat("index-pool").build();


    /**
     * 线程池实现
     * @Author: zlf
     * @date: 2020-08-31
     */
    private static ExecutorService executorService = new ThreadPoolExecutor(
            //线程池默认数量
            corePoolSize,
            //线程池最大数量
            maximumPoolSize,
            //空闲线程存货时间
            keepAliveTime,
            TimeUnit.MILLISECONDS,
            //阻塞队列
            new LinkedBlockingQueue<>(50),
            factory,
            //任务到达上限，做拒绝处理
            new ThreadPoolExecutor.AbortPolicy()
    );

    /**
    * @Description: 获取线程池
    * @Author: zlf
    */
    public static ExecutorService executorService(){
        return executorService;
    }

    /**
     * @Description: 异步执行任务
     * @Author: zlf
     */
    public static void sendTask(Runnable runnable){
        executorService.execute(runnable);
    }
}

# 前端简单实现

  • 附上前端小哥的代码
					  //连接websocket
            connectWs: function () {
                var _this = this;
                // 初始化一个 WebSocket 对象
                var url = urls.javaWs; //java
                //var url = urls.ws; //python
                try {
                    var ws = new WebSocket(url);
                } catch (e) {
                    if (this.wsTimeout) clearTimeout(this.wsTimeout);
                    _this.wsTimeout = setTimeout(function () {
                        _this.connectWs();
                    }, _this.reConnectTimes);
                }

                // 建立 web socket 连接成功触发事件
                ws.onopen = function () {
                    _this.$nextTick(function () {
                        if (_this.curDashboard) {
                            //当websocket链接成功时，给服务端传一个想要哪个合约对的数据的信息，通知服务端响应对应合约对数据
                            var arr = [];
                            for (var i in _this.selectObj) {
                                arr.push(i);
                            }
                            var obj = {
                                op: 'sub', //操作类型
                                group_id: _this.curDashboard.id, //要当前选中仪表盘对应的走势图的数据id
                                group_ids: arr, //要当前选中页的仪表盘的数据
                                ts: _this.interval //分时
                            }
                            obj = JSON.stringify(obj);
                            ws.send(obj);
                        }
                        _this.hasWsOpen = true;
                    })
                };
